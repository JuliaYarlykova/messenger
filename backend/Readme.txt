/api
	GET: Добро пожаловать

/api/Version
	GET: Версия x.x.x

/api/user:
	GET: Получение пользователя-ей (nickname?, id?)
	nickname - "умно" ищет по никнейму,
	id - получает DTO юзера по идентификатору,
	в query пусто - получает список DTO всех пользователей,
	
	POST: Регистрация (nickname, email, passwd, isEmailconfirmed, birthday)
	Никнейм уже есть - 409 + "nickname" в теле,
	Имейл уже есть - 409 + "email" в теле,
	Все норм - перекидывает данные на /login

	DELETE: Удаление пользователя (body - user, query - id)
	Можно перекинуть DTO пользователя в теле, можно его id в query
	Нет токена - 401,
	Не прошел проверку на админа - 403,
	Не прошел проверку на удаление собственного аккаунта - 403,
	Пользователя не существует - 404,
	Все норм - 200

/api/user/login: Авторизация (email, passwd)
	POST:
	Если аккаунт заморожен - 403, 
	Не найдено пользователя с таким имейлом - 404,
	Неверный пароль - 401,
	Все норм - 200 и токен в теле.

/api/user/freeze: Заморозка пользователя (id)
	PUT:
	Нет токена - 401,
	Не прошел проверку на админа - 403,
	Не прошел проверку на заморозку собственного аккаунта - 403,
	Пользователя не существует - 404,
	Все норм - 200

/api/user/password: Смена пароля (id, old, new)
	Нет токена - 401,
	Не прошел проверку на админа - 403,
	Не прошел проверку на смену пароля собственного аккаунта - 403,
	Старый пароль не совпадает - 401 + текст в теле,
	Все норм - 200
